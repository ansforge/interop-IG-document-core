0. Authorization
Définition

Defines the basic properties of every data value. This is an abstract type, meaning that no value can be just a data value without belonging to any concrete type. Every concrete type is a specialization of this general abstract DataValue type.

CourtBase for all types and resources
Contrôle10..1*
Est modificateurfalse
Conteneur logiqueClinicalDocument (CDA Class)
ValidationLes instances de ce type sont validées par templateId

The purpose of a Assigning Authority Name is to assist an unaided human interpreter of an II value to interpret the authority. Définition

The plain code symbol defined by the code system. For example, "784.0" is the code symbol of the ICD-9 code "784.0" for headache.

CourtValeur fixée à 'completed' car on enregistre que les consentements obtenus

Consents referenced in the CDA Header have been finalized (Consent.statusCode must equal "completed") and should be on file.
